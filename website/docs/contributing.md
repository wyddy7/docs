# Контрибьюция

Добро пожаловать! Мы рады, что вы хотите внести вклад в проект лабораторных работ.

## Как внести вклад

### Сообщить об ошибке

Если вы нашли ошибку в документации или в заданиях:

1. Создайте issue в [GitHub репозитории](https://github.com/efremovnv/docs)
2. Опишите проблему подробно
3. Укажите, где именно находится ошибка

### Предложить улучшение

Хотите улучшить существующий материал?

1. Создайте issue с описанием вашего предложения
2. Обсудите его с мейнтейнерами
3. После одобрения создайте Pull Request

### Добавить новую лабораторную работу

1. Создайте папку для лабораторной работы по структуре:
   ```
   docs/labN/
   ├── index.md
   ├── theory.md
   ├── task.md
   ├── solution.md
   └── assets/
   ```

2. Следуйте шаблону из `structure-plan.md`

3. Создайте Pull Request с описанием новой лабораторной работы

## Структура коммитов

Используйте понятные сообщения коммитов:
- `docs: исправлена ошибка в lab1/task.md`
- `feat: добавлена лабораторная работа 3`
- `fix: исправлена ссылка на изображение`

## Вопросы?

Если у вас есть вопросы, создайте issue или свяжитесь с мейнтейнерами проекта.
