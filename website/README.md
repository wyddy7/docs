# Документация

Документация проекта на базе Docusaurus.

## Установка

```bash
npm install
```

## Разработка

```bash
npm start
```

Эта команда запускает локальный сервер разработки и открывает окно браузера. Большинство изменений отражаются в реальном времени без перезагрузки страницы.

## Сборка

```bash
npm run build
```

Эта команда генерирует статический контент в папку `build` и может быть использована для любого хостинга.

## Деплой на GitHub Pages

```bash
npm run deploy
```

Если вы используете GitHub pages для хостинга, эта команда является удобным способом создания сайта и отправки его в ветку `gh-pages`.

## Настройка

Перед использованием необходимо настроить:

1. **GitHub репозиторий** в `docusaurus.config.ts`:
   - `organizationName` - имя вашей организации/пользователя GitHub
   - `projectName` - имя репозитория
   - `url` - URL вашего сайта

2. **Giscus комментарии** в `src/components/Comments/index.tsx`:
   - `repo` - ваш репозиторий GitHub
   - `repoId` - ID репозитория (можно получить через GitHub API)
   - `categoryId` - ID категории для обсуждений

3. **GitHub Stars** в `src/components/GitHubStars/index.tsx`:
   - URL репозитория в fetch запросе
